## 🧠 Technical Features

### 1. Reactive Neural Network
The background utilizes an optimized implementation of `particles.js` to simulate firing synapses:
* **Dynamic Density:** 110 active nodes with automated density scaling.
* **Inference Interaction:** Features a `grab` mode on hover, where nearby neurons physically snap to the cursor to simulate data clustering.

### 2. Live Inference Hero Section
* **Bounding Box Identification:** Mimics a live object detection model identifying the user with a **99.2% confidence score**.
* **System Status Monitoring:** Real-time flickering "REC" dot and scanline animations to maintain the "Active Model" aesthetic.

### 3. Career Route Mapping
* [cite_start]**Chronological Pipeline:** A vertical timeline tracing internships from **IIIT Bhagalpur** to **Angle Two Innovations**[cite: 11, 12, 17, 18].
* [cite_start]**Research Integration:** Highlights specific contributions, such as hybrid YOLOv8+CNN models and **IEEE CICN 2026** submissions[cite: 13, 15].
